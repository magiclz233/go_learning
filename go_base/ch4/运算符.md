## 位运算符

```go
& 按位与 参与运算的两个数的二进制相与  即 11 = 1; 10,01,00 = 0
| 按位或 二进制相或 即 11,10,01 = 1; 00 = 0
^ 异或 两个位相同为0,不同为1 10,01 = 1; 11,00 = 0
<< 左移 1 << 2  即数的二进制左移2位 0001 -> 0100 = 8 
>> 右移 和左移相反

//go独有
&^ 按位置零
1 &^ 0 = 1
1 &^ 1 = 0
0 &^ 1 = 0
0 &^ 0 = 0
```

&^ 的功能是将运算符左边数据相异的位保留，相同位清零。

此运算符特点:

1、如果右侧是0，则左侧数保持不变

2、如果右侧是1，则左侧数一定清零

3、功能同a&(^b)相同

4、如果左侧是变量，也等同于：

var a int

a &^= b
